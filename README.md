# Домашнее задание: Декораторы в Python

Реализация декораторов для логирования вызовов функций.

## Задания

### 1. Базовый декоратор
Декоратор `logger` записывает логи в файл `main.log`

### 2. Параметризованный декоратор  
Декоратор `logger` записывает логи в указанный файл

### 3. Применение декоратора
Использование декоратора для функций работы с кулинарной книгой

## Файлы проекта

- `task1_decorator.py` - базовый декоратор и тест 1
- `task2_decorator.py` - параметризованный декоратор и тест 2  
- `task3_application.py` - применение декоратора и тест 3
